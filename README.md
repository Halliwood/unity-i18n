# unity-i18n
A great tool for unity i18n.

## usage

```
npm i unity-i18n -g
```

for unity project:
```
unity-i18n -s "G:/dldl_kr/trunk/" -o "G:/dldl_kr/trunk/tools/i18n/dictionary/tw" -d "unity" -S
```

for laya project:
```
unity-i18n -s "G:/dldl_kr/trunk/" -o "G:/dldl_kr/trunk/tools/i18n/dictionary/tw" -d "laya" -S
```
